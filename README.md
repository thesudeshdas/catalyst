## Run Locally

> ‚ùóMake sure to install the [Backend](https://github.com/thesudeshdas/catalyst-backend) prior to the following steps.

1 - Clone the project

```bash
  git clone https://github.com/thesudeshdas/catalyst.git
```

2 - Go to the project directory

```bash
  cd catalyst
```

3 - Install dependencies

```bash
  npm install
```

4 - Add environment variables

```bash
  REACT_APP_POSTS_API_URL
  REACT_APP_AUTH_URL
  REACT_APP_CLOUDINARY_URL
```

5 - Start the server

```bash
  npm run start
```

## Environment Variables

> üí° To run this project, you will need to add the following environment variables to your .env file

`REACT_APP_POSTS_API_URL` -
The URL for `/posts` API (ex - http://localhost:8000/posts)

`REACT_APP_AUTH_URL` - The URL for `/auth` API (ex - http://localhost:8000/users)

`REACT_APP_CLOUDINARY_URL` - The URL for cloudinary where images will be uploaded
